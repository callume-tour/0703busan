DAY 1 · 7/03

太宗台、海雲台與青沙浦海岸線

桃園機場 / 釜山 → Hill Spa 汗蒸幕 → 太宗台 DANUBI 卡通列車遊園 → THE BAY101 → 海雲台海邊沙灘 → 天空膠囊列車 → 青沙浦踏石天空步道 → 青沙浦最美鐵道 + 紅白燈塔 → 飯店

Hill Spa 汗蒸幕 活動

以汗蒸與海水溫泉作為抵達後的恢復站。

#必拍 海景汗蒸幕

DANUBI 卡通列車 交通 / 景點

串起太宗台觀景點、燈塔與海崖步道。

#必拍 卡通列車

THE BAY101 景點 / 食物

海雲台高樓群、水面倒影與城市夜景。

#必拍 海雲台夜景

天空膠囊列車 活動 / 交通

沿海雲台海岸慢行，是近年釜山人氣體驗。

#必拍 彩色膠囊列車

青沙浦踏石天空步道 景點

伸向海面的觀景步道，可近距離看浪花與礁岩。

#必拍 玻璃步道

奶奶家豬肉湯 / 鍋巴人蔘雞 食物

豬肉湯飯是釜山庶民味，人蔘雞是韓國夏季補身代表。

#必吃 豬肉湯 #必吃 人蔘雞

餐食 / 飯店：午餐：奶奶家豬肉湯｜晚餐：鍋巴人蔘雞 이우철한방누룽지 삼계탕｜飯店：ARBAN CITY HOTEL
導遊秘笈：太宗台以海蝕崖與松林聞名；海雲台與青沙浦則是釜山最經典的海岸觀光帶。
👗 建議穿著 + ☀️ 天氣提示：紅眼班機後行程較滿，穿輕便透氣衣物與好走鞋；建議帶防曬、帽子、薄外套與小雨傘。

DAY 2 · 7/04

速度感、Outlet 與海邊古寺

SKY LUGE 渠道滑車 + 高空滑索 Hyfly + 景觀纜車（兩次體驗券） → 樂天 Outlet 購物中心 → 海東龍宮寺 → 飯店

SKY LUGE 渠道滑車 活動

可自行控制速度，動感但不會太刺激。

#必玩 渠道滑車

Hyfly + 景觀纜車 活動 / 交通

高空視角俯瞰海岸與園區景色。

#必拍 高空視角

樂天 Outlet 購物

品牌、運動用品與伴手禮集中採買。

#必買 韓系服飾 #必買 美妝小物

海東龍宮寺 景點

依岩而建、面向東海，是韓國少見的海岸寺院。

#必拍 海上寺廟

餐食 / 飯店：午餐：元祖鮑魚粥｜晚餐：味贊王烤肉，四份五花肉 + 每人一份鍋飯 + 2 份大醬湯｜飯店：ARBAN CITY
導遊秘笈：上午打開旅行節奏，下午採買，最後以面海而建的海東龍宮寺收尾。
👗 建議穿著 + ☀️ 天氣提示：穿包覆性球鞋，避免長裙、拖鞋與鬆散配件；請補水並準備防曬。

松島海上纜車（含來回纜車票） → 松島龍宮雲橋（韓國觀光 100 選） → 積木村（甘川洞文化村） → 西面活動 → 飯店

松島海上纜車 活動 / 交通

#必拍 海上纜車

松島龍宮雲橋 景點

海上步道可看見礁岩、海浪與碧綠海面。

#必拍 360 度海景

甘川洞文化村 景點

山坡聚落經藝術改造後成為彩色村落。

#必拍 小王子 #必拍 彩色山城

西面活動 購物 / 食物

地下街、美妝店、餐廳與咖啡店密集。

#必買 美妝 #必吃 宵夜

餐食 / 飯店：午餐：烤魚韓定食 + 石鍋飯 + 大醬湯｜晚餐：自理｜飯店：NORTH HARBOR HOTEL
導遊秘笈：松島海景與甘川洞彩色房舍，是釜山照片最容易出片的組合。
👗 建議穿著 + ☀️ 天氣提示：松島與甘川洞都有步行與坡道，建議穿止滑好走鞋。

注意事項

1. 行李重量：托運行李以 1 大件合計不超過 20 公斤、手提行李 1 小件合計不超過 10 公斤為原則。
2. 廉航服務：廉價航空無提供機上餐食、毛毯、水與其他機上服務，請自行攜帶。
3. 行動電源：必須隨身上機，單獨放入透明夾鏈袋，並做好防短路措施。
4. 韓國電壓：韓國為 220-240V，請自備變壓器及萬用插頭。
5. 入境申報：免稅香煙 200 支、酒 1 瓶；外幣與貴重物品須詳細填寫。
6. 盥洗用品：部分酒店不提供個人盥洗用具，牙刷、牙膏等請自行攜帶。
7. 保險提醒：已投保履約責任險 250 萬及意外醫療險 20 萬；建議自行投保旅平險。
8. 返國限制：不可攜帶任何形式肉製品，真空包裝也不可，亦不可攜帶蔬果。
9. 行程調整：若受航班、天候、交通等因素影響，得依實際狀況調整順序。
